# gHipster

## What's inside
  
  * Gradle
  * Spring boot
  * Spock test framework

  * Grunt
  * AngularJS
  * Karma/Jasmine unit tests

Frontend was generated by yeoman generator-angular


## Run developement mode

  1. gradle bootRun

or

with `cd frontend`
  2. gradle grunt_build


## Build
  1. gradle build
